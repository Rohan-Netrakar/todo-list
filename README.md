## Features

- Add new tasks with a descriptive input field.
- Mark tasks as completed with a checkbox.
- Edit task descriptions directly on the list.
- Delete tasks when they are no longer needed.
- Data persistence using local storage to save tasks between sessions.
- Responsive design for both desktop and mobile devices.

## Setup and Installation

To use this To-Do List application locally, follow these steps:

1. **Clone the repository:**

    ```bash
    git clone https://github.com/Rohan-Netrakar/todo-list.git
    ```

2. **Navigate to the project directory:**

    ```bash
    cd todo-list
    ```

3. **Open the `index.html` file in your preferred web browser to start using the app.**

## How to Use

1. Open the To-Do List application in your web browser.
2. Enter a new task into the input field and click the "Add" button.
3. Each task will appear in the list with options to mark as completed, edit the task description, or delete the task.
4. Mark tasks as completed by clicking on the checkbox.
5. Edit task descriptions by clicking on the task text and making changes directly.
6. Delete tasks by clicking the "Delete" button next to each task.
7. Your tasks will be saved locally, so you can close the app and come back to it later without losing your data.
